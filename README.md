# 🏡 Luxury Housing Analytics Dashboard – Bangalore Market Insights

## 📌 Project Overview

This project analyzes the **Bangalore luxury housing market** using an end-to-end data analytics workflow.

It combines:
- **Python (data cleaning)**
- **MySQL (data storage)**
- **Power BI (visualization)**

to generate meaningful business insights for real estate decision-making.

The dashboard provides insights into:
- Market performance
- Builder efficiency
- Booking conversion trends
- Customer preferences
- Geographical distribution of projects

---

## 🎯 Business Objective

To build an **interactive Power BI dashboard** that helps stakeholders:
- Identify top-performing builders
- Analyze booking trends across time and locations
- Understand customer behavior and preferences
- Improve conversion rates and sales strategies

---

## 🛠️ Tools & Technologies Used

- **Python** → Data Cleaning & Preprocessing  
- **MySQL** → Database Storage & Querying  
- **Power BI** → Data Visualization & Dashboarding

## 📌 Key Insights Summary

- High revenue ≠ high ticket size → **volume matters**
- Booking conversion is relatively low (~33%)
- Referral-based sales channels are most effective
- Amenities positively influence buyer decisions
- Market demand varies across quarters and locations
